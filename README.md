# ULake

This is the USTH Data Lake.


# Boot it up

To build everything:

```bash
./gradlew quarkusBuild
```

To start the lake services, use

```bash
./gradlew quarkusDev
```

# Hacking

We use Intellij IDEA as the main IDE. Simply open this directory in IDEA and start the available sub-projects as services.
